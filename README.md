## Installation & Setup
1. Install [Hugo](https://gohugo.io/getting-started/installing)
2. Run a Developement Server

   ```sh
   hugo server -D
   ```

## Building & Running for Production
1. Install [Hugo](https://gohugo.io/getting-started/installing)
2. Build a static website

   ```sh
   hugo build --minify
   ```
